# Chatbot with Persistent Memory & RAG

A lightweight chatbot supporting:
- Local or cloud LLM inference
- Persistent conversation memory (SQLite)
- Retrieval-Augmented Generation (RAG)
- Dockerized setup for reproducibility

## Features
- UI created using streamlit
- Switchable inference (local / cloud)
- SQLite-backed chat memory
- Document-aware responses via RAG
- Docker & Docker Compose support

```bash
python app/main.py
